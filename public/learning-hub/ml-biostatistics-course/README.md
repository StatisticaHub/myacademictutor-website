# Machine Learning in Biostatistics

A structured R-based course on machine learning for medical statistics, clinical prediction modelling, survival analysis, and biomedical data science.

## Website

Access the course here:

[Machine Learning in Biostatistics](https://rahulbalwan.github.io/ml-biostatistics-course/)

## About

This course explains machine learning through a **medical statistics and biostatistics lens**.

The focus is not only on algorithms, but on how models should be developed, validated, interpreted, and reported in medical research.

Key themes include:

- prediction versus inference;
- validation and generalisation;
- overfitting and data leakage;
- discrimination and calibration;
- clinical usefulness;
- reproducible R workflows;
- transparent reporting of medical prediction models.

## Current content

### Module 1: Foundations

- What is machine learning in biostatistics?
- Prediction vs inference
- Validation, overfitting, and data leakage

### Module 2: Supervised Learning

- Supervised learning for medical data
- Logistic regression for clinical prediction
- Classification metrics in medicine
- R implementation examples

### Case Studies

- Diabetes risk prediction using logistic regression
- ROC curve and AUC interpretation
- Calibration plot interpretation
- Clinical interpretation of threshold-based metrics
