Course Roadmap

This roadmap shows the planned structure of the course.

The course is built around a simple learning pathway:

Concept → Method → R implementation → Clinical interpretation → Case study

Module 1: Foundations

1. What is Machine Learning in Biostatistics?

Topics:

  • machine learning in medical research;
  • prediction versus explanation;
  • statistical learning;
  • clinical prediction models;
  • why medical machine learning is different from general machine learning.

Materials:

  • lecture;
  • note;
  • exercise.

2. Validation, Overfitting and Data Leakage

Topics:

  • training error versus test error;
  • overfitting;
  • validation;
  • internal validation;
  • external validation;
  • data leakage;
  • why leakage is dangerous in medical prediction.

Materials:

  • lecture;
  • note;
  • exercise.
