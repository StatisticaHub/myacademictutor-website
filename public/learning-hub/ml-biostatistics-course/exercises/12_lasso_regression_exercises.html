Part H: Short written answer

Question 19

Write a short explanation of lasso regression for a clinician who wants a smaller prediction model.

Suggested answers

Answer 1

Lasso regression is a regularisation method that shrinks coefficients. Some coefficients can be shrunk exactly to zero, which removes those predictors from the model. This means lasso can perform feature selection while building a prediction model.

Answer 2

B. It can shrink some coefficients exactly to zero.

Answer 3

A sparse model uses a smaller number of predictors. It may be attractive in clinical prediction because it can be easier to measure, explain, implement, and use in practice.

Answer 4

Lasso selected those predictors because they helped prediction in that dataset and modelling workflow. This does not prove that they are the only clinically important predictors or that they cause the outcome.

Answer 5

  1. Retained predictors: glucose, BMI, age.
  2. Removed predictors: blood pressure, triceps thickness.
  3. This may create a smaller and more practical model.
  4. We should be cautious because selected predictors may be unstable, dataset-specific, or not externally validated.

Answer 6

Ridge shrinks coefficients but usually keeps all predictors. Lasso can shrink some coefficients exactly to zero, so it can remove predictors and perform feature selection.

Answer 7

Small lambda  → less shrinkage → more predictors retained
Large lambda  → more shrinkage → fewer predictors retained

Answer 8

If lambda is too large, lasso may remove too many predictors and underfit the data.

Answer 9

If lambda is too small, lasso may keep too many predictors and may not sufficiently reduce overfitting.

Answer 10

No. If the biomarkers are highly correlated, lasso may choose one of them somewhat arbitrarily. Biomarker A may be useful for prediction, but this does not prove it is biologically more important.

Answer 11

When predictors are strongly correlated, several predictors may contain similar information. Lasso may choose one and remove others, but small changes in the data may change which predictor is selected.

Answer 12

Elastic net may be considered because it combines lasso and ridge behaviour and can handle correlated predictors more smoothly.

Answer 13

The full dataset was used to select predictors before cross-validation. This means the validation folds influenced feature selection. The reported cross-validated AUC may be optimistic.

Answer 14

Correct workflow:

For each cross-validation fold:
    1. Use only the training fold.
    2. Run lasso inside the training fold.
    3. Select predictors inside the training fold.
    4. Fit the model inside the training fold.
    5. Evaluate on the validation fold.

Answer 15

With many predictors, some will appear related to the outcome by chance. If the full dataset is used to select features, random signals from validation data can enter model development and inflate performance.

Answer 16

They should ask:

  • Were biomarkers selected inside validation?
  • Are selected biomarkers stable across resamples?
  • Are they biologically plausible?
  • Do they validate externally?
  • Could batch effects explain the selection?
  • Are they clinically measurable?

Answer 17

They should report calibration, Brier score, calibration slope/intercept, decision curve analysis if relevant, and external validation if available.

Answer 18

External validation tests whether the selected biomarkers and model performance generalise to new patients, settings, platforms, or populations.

Answer 19

Lasso regression builds a prediction model while also reducing the number of predictors. It shrinks weak coefficients to zero, so some variables are removed. This can make the model smaller and more practical, but the selected predictors still need validation before clinical use.
