Exercise 7: Bootstrap Validation

Learning aim

This exercise helps you understand apparent performance, optimism, and the role of bootstrap validation in clinical prediction modelling.

By the end of this exercise, you should be able to explain:

  • what apparent performance means;
  • why apparent performance is usually too optimistic;
  • what bootstrap sampling means;
  • how optimism-corrected performance is interpreted;
  • why bootstrap validation does not replace external validation.

Question 1: Concept check

What is apparent performance?

Show suggested answer Apparent performance is the performance of a model when it is evaluated on the same data used to develop or fit the model. For example: ```text Fit the model on the full development dataset. Evaluate the model on the same full development dataset. ``` This is usually optimistic because the model has already seen the data.

Question 2: Optimism

A model has:

Apparent AUC = 0.86
Estimated optimism = 0.05

What is the optimism-corrected AUC?

Show suggested answer The optimism-corrected AUC is: ```text 0.86 - 0.05 = 0.81 ``` So the optimism-corrected AUC is approximately: ```text 0.81 ```

Question 3: Interpretation

Why is apparent performance usually too optimistic?

Show suggested answer Apparent performance is usually too optimistic because the model is evaluated on the same data it used to learn. The model may capture: - real signal; - random noise; - dataset-specific patterns; - accidental associations. These patterns may not generalise to new patients.

Question 4: Bootstrap sample

What does it mean to sample with replacement?

Show suggested answer Sampling with replacement means that after a patient is selected, that patient is placed back into the pool before the next selection. As a result: - some patients may appear more than once in a bootstrap sample; - some patients may not appear at all; - the bootstrap sample usually has the same size as the original dataset.

Question 5: Clinical interpretation

A cancer recurrence prediction model has:

Apparent C-index = 0.78
Optimism-corrected C-index = 0.70

What does this suggest?

Show suggested answer This suggests that the apparent performance was overly optimistic. The model performed well on the development data, but bootstrap validation estimated that some of this performance may not generalise to new patients. The optimism-corrected C-index of 0.70 is a more realistic internal estimate than 0.78.

Question 6: External validation

A model has good optimism-corrected performance after bootstrap validation.

Is it ready for clinical use?

Show suggested answer Not necessarily. Bootstrap validation is internal validation. It estimates performance within the development dataset or source population. Before clinical use, the model would still need: - external validation; - calibration assessment; - clinical usefulness evaluation; - assessment of predictor availability; - subgroup and fairness checks; - transparent reporting.

Question 7: Method choice

Why might bootstrap validation be attractive in studies with limited sample size?

Show suggested answer Bootstrap validation can be attractive because it allows the full dataset to be used for model development while still estimating optimism. Unlike a simple train/test split, it does not permanently hold out a large portion of the data. This can be useful when clinical datasets are small.

Reflection

Why is it risky to report only apparent performance in a medical prediction study?

Show reflection answer It is risky because apparent performance may make the model look better than it really is. If a model is overfitted, it may appear accurate in the development data but fail on new patients. In clinical settings, this can lead to unsafe decisions, poor risk communication, and misleading claims about model usefulness.
