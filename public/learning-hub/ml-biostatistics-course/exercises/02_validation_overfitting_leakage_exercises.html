Exercise 2: Validation, Overfitting and Data Leakage

Exercise 1: Training error vs test error

A model for predicting hospital mortality gives the following results:

Training AUC = 0.96
Test AUC = 0.69

Answer:

  1. Which performance estimate is more relevant for future patients?
  2. What does the large difference suggest?
  3. Give two possible reasons for this pattern.
Show answer The test AUC is more relevant because it estimates performance on data not used to train the model. The large drop from training to test performance suggests overfitting or possible leakage. It may also happen if the test population differs from the training population or if the training dataset was too small.

Exercise 2: Identify overfitting

Which of the following are signs of possible overfitting?

Scenario Possible overfitting?
Training accuracy 99%, test accuracy 65%  
Training AUC 0.80, test AUC 0.78  
Very complex model fitted to 80 patients with 500 predictors  
Model performance reported only on training data  
External validation performance similar to internal validation  
Show answer Possible overfitting: training accuracy 99% with test accuracy 65%; very complex model with 80 patients and 500 predictors; reporting only training performance. Similar internal and external validation performance is reassuring, though still needs careful interpretation.

Exercise 3: Bias or variance?

Classify each situation as mainly high bias or high variance.

Situation High bias or high variance?
A simple linear model misses a strong nonlinear effect  
A very deep decision tree performs perfectly on training data but poorly on test data  
A logistic regression model underfits a complex image-based problem  
A model changes substantially when trained on a slightly different sample  
Show answer A simple linear model missing nonlinear effects is high bias. A deep tree performing well on training data but poorly on test data is high variance. Logistic regression underfitting an image-based problem is high bias. A model that changes substantially between samples is high variance.

Exercise 4: Internal or external validation?

Classify each validation approach.

Validation approach Internal or external validation?
5-fold cross-validation in the development dataset  
Bootstrap validation in the same cohort  
Testing the model in another hospital  
Testing the model in the same hospital but later calendar years  
Testing the model in another country  
Show answer 5-fold cross-validation and bootstrap validation are internal validation. Testing in another hospital or country is external validation. Testing in the same hospital but later years may be considered temporal external validation or temporal validation, depending on the study design.

Exercise 6: Leakage from preprocessing

A researcher does the following:

  1. Uses the full dataset to standardise all predictors.
  2. Splits the data into training and test sets.
  3. Fits a model on the training set.
  4. Evaluates performance on the test set.

Question:

  1. What is wrong with this workflow?
  2. What is the correct workflow?
Show answer The researcher used information from the test set during standardisation. This causes data leakage. The correct workflow is to split the data first, estimate standardisation parameters using the training data only, and then apply those parameters to the test data.

Exercise 7: Leakage from feature selection

A researcher has 10,000 gene expression predictors and 300 patients.

Workflow:

  1. Selects the 100 genes most associated with outcome using the full dataset.
  2. Performs 10-fold cross-validation using only those 100 genes.
  3. Reports high cross-validated AUC.

Question:

  1. Why is this problematic?
  2. What should have been done instead?
Show answer This is problematic because the feature selection used outcome information from the full dataset before cross-validation. The cross-validation folds are no longer independent. Feature selection should be performed inside each training fold, and performance should be evaluated on held-out folds that were not used for feature selection.

Exercise 9: Calibration problem

A model predicts 10-year cardiovascular risk.

Among patients predicted to have 30% risk, the observed event rate is 12%.

Answer:

  1. Is the model well calibrated in this risk group?
  2. Is the model overestimating or underestimating risk?
  3. Why could this be clinically harmful?
Show answer The model is not well calibrated in this group. It is overestimating risk because predicted risk is 30% but observed risk is 12%. This could lead to unnecessary treatment, anxiety, additional testing, or inappropriate risk communication.

Exercise 11: Mini case study

A team develops a model to predict 1-year mortality after cancer diagnosis.

They report:

Sample size: 150 patients
Predictors: 2,000 biomarkers
Model: random forest
Training AUC: 0.99
No test set
No cross-validation
No calibration plot
No external validation

Answer:

  1. List at least five concerns.
  2. What validation approach would you recommend?
  3. What additional reporting would you request?
Show answer Concerns include small sample size, many predictors, high risk of overfitting, no internal validation, no external validation, no calibration assessment, training AUC only, and unclear feature selection. At minimum, cross-validation or bootstrap validation should be used, ideally with feature selection inside resampling. External validation in an independent cohort would be preferred. Reporting should include outcome definition, predictor definitions, missing data handling, model tuning, calibration, discrimination, and limitations.
