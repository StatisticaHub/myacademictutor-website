Exercise 5: Confusion matrix calculation

A model predicts disease status for 100 patients.

Results:

True positives = 30
False positives = 10
True negatives = 50
False negatives = 10

Calculate:

  1. Accuracy
  2. Sensitivity
  3. Specificity
  4. Positive predictive value
  5. Negative predictive value
Show answer Total = 30 + 10 + 50 + 10 = 100 Accuracy: ```text (TP + TN) / Total = (30 + 50) / 100 = 0.80 ``` Sensitivity: ```text TP / (TP + FN) = 30 / (30 + 10) = 0.75 ``` Specificity: ```text TN / (TN + FP) = 50 / (50 + 10) = 0.833 ``` Positive predictive value: ```text TP / (TP + FP) = 30 / (30 + 10) = 0.75 ``` Negative predictive value: ```text TN / (TN + FN) = 50 / (50 + 10) = 0.833 ```

Exercise 6: Accuracy problem

A disease has prevalence 2%.

A model predicts every patient as disease negative.

Answer:

  1. What is the accuracy?
  2. What is the sensitivity?
  3. Why is this model clinically useless?
  4. Which metrics would be more informative?
Show answer Accuracy is 98%, because 98% of patients do not have the disease. Sensitivity is 0%, because the model detects none of the true disease cases. The model is clinically useless for identifying disease because it classifies every patient as negative. More informative metrics include sensitivity, specificity, PPV, NPV, ROC AUC, precision-recall curve, and calibration. In screening, sensitivity is especially important.

Exercise 7: Threshold choice

A model gives predicted probability of disease.

For screening, should we usually prefer:

  1. a lower threshold with higher sensitivity, or
  2. a higher threshold with higher specificity?

Explain your answer.

Show answer For screening, we often prefer a lower threshold with higher sensitivity because the goal is to avoid missing true cases. However, this usually increases false positives. Whether this is acceptable depends on the harm, cost, and burden of follow-up testing.

Exercise 8: PPV and prevalence

Two populations are screened using the same model.

The model has the same sensitivity and specificity in both populations.

Population A has disease prevalence 20%.

Population B has disease prevalence 2%.

Question:

In which population will the positive predictive value usually be higher?

Show answer The positive predictive value will usually be higher in Population A because disease prevalence is higher. PPV depends strongly on prevalence. When disease is rare, many positive predictions may be false positives even if sensitivity and specificity are reasonable.

Exercise 9: AUC interpretation

A model has:

AUC = 0.82

Answer:

  1. What does this tell us?
  2. Does this tell us whether predicted probabilities are well calibrated?
  3. What else should be assessed?
Show answer An AUC of 0.82 suggests the model has reasonably good discrimination. It can rank patients with the outcome higher than patients without the outcome fairly well. AUC does not tell us whether predicted probabilities are calibrated. Calibration should also be assessed, along with clinical usefulness, sensitivity, specificity, PPV, NPV, and performance in relevant subgroups.

Exercise 10: Logistic regression as baseline

Why is logistic regression often a good first model for binary clinical prediction?

Write 4–6 sentences.

Show answer Logistic regression is a good baseline model because it is interpretable, familiar in medical statistics, and produces predicted probabilities. It works well for many tabular clinical datasets and can be assessed using standard prediction metrics. It is also easier to validate and report than many complex models. More complex machine learning models should usually be compared against logistic regression to show whether they provide meaningful improvement.

Exercise 12: Mini case study

A team develops a model to predict 30-day readmission.

They use the following variables:

Age
Sex
Admission diagnosis
Comorbidities
Length of stay
Discharge medications
Number of previous admissions
Discharge destination
Readmission status

The model is intended to be used at hospital discharge.

Answer:

  1. Which variables are appropriate?
  2. Which variable should not be used as a predictor?
  3. Is length of stay appropriate in this case?
  4. Would your answer change if the prediction time point was hospital admission?
Show answer Appropriate predictors include age, sex, admission diagnosis, comorbidities, length of stay, discharge medications, number of previous admissions, and discharge destination, assuming the prediction is made at discharge. Readmission status should not be used as a predictor because it is the outcome. Length of stay is appropriate if the prediction is made at discharge because it is known at that time. If the prediction time point was hospital admission, length of stay, discharge medications, and discharge destination would not be available and would become leakage-risk variables.
