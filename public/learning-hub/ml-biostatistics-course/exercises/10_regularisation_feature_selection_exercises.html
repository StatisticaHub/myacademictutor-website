Exercise 10: Regularisation and Feature Selection

Learning aim

This exercise helps you understand why regularisation is useful when there are many predictors and why feature selection must be interpreted carefully.

By the end of this exercise, you should be able to explain:

  • why many predictors increase overfitting risk;
  • how ridge regression differs from lasso regression;
  • why elastic net can be useful for correlated predictors;
  • why feature selection must be included inside validation;
  • why selected features should not automatically be interpreted as causal.

Question 1: Concept check

Why does overfitting become more likely when there are many predictors and few patients?

Show suggested answer When there are many predictors compared with the number of patients, the model has many opportunities to find patterns that are due to noise rather than real signal. This can lead to excellent performance in the development data but poor performance in new patients.

Question 2: Ridge regression

What does ridge regression do to model coefficients?

Show suggested answer Ridge regression shrinks coefficients towards zero. It usually keeps all predictors in the model, but reduces their size. This can improve stability, especially when predictors are correlated.

Question 3: Lasso regression

How is lasso regression different from ridge regression?

Show suggested answer Lasso regression can shrink some coefficients exactly to zero. This means lasso can perform variable selection. Ridge regression shrinks coefficients towards zero but usually keeps all predictors in the model.

Question 4: Elastic net

Why might elastic net be useful for omics data?

Show suggested answer Omics data often contain many correlated predictors. Elastic net combines ridge and lasso penalties. This means it can: - shrink coefficients; - perform some feature selection; - handle correlated predictors more stably than lasso alone.

Question 5: Interpretation

Why should selected features from a lasso model not automatically be interpreted as causal factors?

Show suggested answer Lasso selects features that help prediction in a specific dataset. Selection may be affected by: - correlation between predictors; - sample size; - random noise; - tuning choices; - measurement artefacts. A selected feature may improve prediction without being causal.

Question 6: Validation

Why is it problematic to perform feature selection before train/test splitting?

Show suggested answer If feature selection is performed before train/test splitting, the test set has already influenced which features were selected. This creates data leakage. The test set is no longer truly independent, and performance estimates may be too optimistic.

Question 7: Clinical reasoning

A lasso model selects a biomarker for cancer recurrence prediction.

What should you check before claiming that this biomarker is clinically important?

Show suggested answer You should check: - whether the biomarker is biologically plausible; - whether it is selected consistently across resamples; - whether it improves validated prediction performance; - whether it is available and measurable in practice; - whether the finding is externally validated; - whether it may simply be correlated with another important predictor.

Question 8: Many predictors

You have:

100 patients
20,000 gene expression features

Why would ordinary logistic regression be problematic?

Show suggested answer Ordinary logistic regression would be problematic because the number of predictors is far larger than the number of patients. This can lead to unstable estimates, overfitting, and failure to generalise. Regularisation or dimension reduction would usually be needed.

Question 9: Prediction versus explanation

A regularised model selects a group of methylation sites that predict exposure status.

Does this prove that those methylation sites are caused by the exposure?

Show suggested answer No. The model identifies features that help prediction. It does not automatically establish causality. The selected methylation sites may be associated with exposure, correlated with other features, affected by confounding, or selected due to dataset-specific patterns.

Reflection

In a biomedical prediction model, how would you balance statistical feature selection with clinical or biological reasoning?

Show reflection answer A strong modelling strategy should combine statistical performance with subject-matter reasoning. Statistical feature selection can help identify predictors that improve prediction, but clinical and biological reasoning is needed to assess whether selected features are meaningful, measurable, stable, and practical. Validation is essential before making strong claims about selected features.
