How to use the notes

A good learning pathway is:

  1. Read the lecture first.
    Use the lecture page to understand the intuition and clinical motivation.

  2. Study the note carefully.
    Use the note for definitions, equations, assumptions, interpretation and deeper statistical explanation.

  3. Complete the exercise page.
    Check whether you can apply the concept without only memorising terminology.

  4. Run the R code.
    Connect the theory to implementation and interpretation.

  5. Work through the case study.
    See how the concept appears in a realistic medical prediction workflow.

Note style

The notes are written in a more academic style than the lectures.

Each note usually includes:

  • the main statistical idea;
  • why the method is useful in medical research;
  • mathematical or conceptual foundations;
  • interpretation of model output;
  • common mistakes;
  • links to exercises, code and case studies where relevant.

The goal is to help learners move from intuition to careful statistical understanding.
