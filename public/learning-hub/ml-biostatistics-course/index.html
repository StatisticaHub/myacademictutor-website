Machine Learning in Biostatistics

A structured R-based course on machine learning for medical statistics, clinical prediction modelling, survival analysis, and biomedical data science.

Course overview

Machine Learning in Biostatistics is designed for learners who want to understand machine learning through a medical statistics and biostatistics lens.

The course focuses on:

  • prediction versus inference;
  • validation and generalisation;
  • overfitting and data leakage;
  • discrimination and calibration;
  • clinical usefulness;
  • regularisation and feature selection;
  • transparent reporting;
  • reproducible R workflows;
  • applied clinical case studies.

In medical machine learning, high accuracy is not enough. A useful model should be valid, calibrated, clinically interpretable, reproducible, clinically useful, and generalisable to new patients or populations.

Learning pathway

A recommended way to use this course:

  1. Read the lecture page
    Start with the story-based lecture to understand the intuition, common doubts, and clinical motivation.

  2. Read the detailed note
    Use the notes for deeper theory, statistical explanation, and examples.

  3. Complete the exercises
    Test your conceptual understanding and applied reasoning.

  4. Run the R code
    Implement the method and interpret the results.

  5. Work through the case study
    Connect the method to a realistic medical prediction problem with figures and clinical interpretation.

Course format

This website is designed as a structured written learning resource.

The lectures use a conversational teaching style to make the ideas more intuitive and memorable. The notes remain more detailed and academic. The exercises, R scripts, and case studies help connect the concepts to practical medical machine learning.
