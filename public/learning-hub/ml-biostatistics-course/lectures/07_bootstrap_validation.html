Today’s big question

How can bootstrap validation help estimate optimism in model performance?

Bootstrap validation is an internal validation method.

It repeatedly samples from the development data, fits the model, and estimates how much apparent performance may be too optimistic.

Intuition first

When a model is evaluated on the same data used to build it, it usually looks better than it really is.

This is called optimism.

Bootstrap validation tries to estimate that optimism.

Bootstrap validation helps adjust for overly optimistic performance estimates from the development data.

What is a bootstrap sample?

A bootstrap sample is created by sampling patients from the original dataset with replacement.

This means:

  • some patients appear more than once;
  • some patients do not appear at all;
  • the sample has the same size as the original dataset.

The patients not selected in a bootstrap sample are sometimes called out-of-bag observations.

Apparent performance

Apparent performance means performance calculated on the same dataset used to fit the model.

For example:

Fit model on all development data
Evaluate model on the same data

This often gives an overly positive result.

It is like giving a student the same exam paper they practised on.

Optimism

Optimism is the difference between performance on the data used for model development and performance on new data.

Conceptually:

Optimism = apparent performance - expected new-data performance

Bootstrap validation estimates this optimism and can be used to correct the apparent performance.

Medical example

Suppose a logistic regression model for cancer recurrence has:

Apparent AUC = 0.84

After bootstrap validation, estimated optimism is:

Optimism = 0.04

Then optimism-corrected AUC is approximately:

0.84 - 0.04 = 0.80

This gives a more realistic estimate of performance.

Common mistake

Common mistake: reporting only apparent performance

If you fit a model and report performance on the same data, the result is likely too optimistic.

Medical prediction models should include internal validation, such as bootstrap validation or cross-validation.

Clinical reality check

Clinical reality check

An optimism-corrected model can still fail externally.

Bootstrap validation improves internal assessment, but it does not prove the model will work in another hospital or population.

External validation is still needed before clinical use.

Key takeaway

Bootstrap validation is useful for estimating optimism in model performance.

It is especially valuable in clinical prediction modelling because it uses the full development dataset while still assessing how much performance may be exaggerated.

Reflection question

Why is apparent performance usually too optimistic?

Show suggested answer Because the model is evaluated on the same data it used to learn. It may capture noise or dataset-specific patterns, so performance on the development data is usually better than expected performance on new patients.
