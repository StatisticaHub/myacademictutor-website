Lasso Regression

Big idea

Lasso regression is a regularisation method that can shrink some coefficients exactly to zero.

This means lasso can perform feature selection.

In medical prediction, lasso can be useful when we have many candidate predictors and want a smaller model.

But selected predictors should be interpreted carefully.

Scene: too many predictors

Learner: I have 60 possible predictors for a clinical prediction model. Ordinary logistic regression feels too crowded.

Dr. Clinic: Also, collecting 60 predictors for every patient may not be realistic.

Prof. Stat: This is where lasso can help. It can shrink some coefficients to zero and produce a smaller model.

Learner: So lasso chooses the important predictors?

Prof. Stat: It chooses predictors useful for prediction in the data. That is not the same as proving clinical or causal importance.

1. What lasso regression does

Lasso regression adds a penalty to the model-fitting process.

Like ridge, it shrinks coefficients.

But lasso can shrink some coefficients exactly to zero.

If a coefficient becomes zero, that predictor is removed from the model.

This makes lasso useful for feature selection.

2. A simple example

Suppose ordinary logistic regression gives:

glucose coefficient = 1.10
BMI coefficient = 0.70
age coefficient = 0.40
pressure coefficient = 0.12
triceps coefficient = 0.05

Lasso may give:

glucose coefficient = 0.90
BMI coefficient = 0.55
age coefficient = 0.30
pressure coefficient = 0
triceps coefficient = 0

In this example, pressure and triceps are removed.

The final model is smaller.

3. Why lasso is attractive in medical prediction

Lasso can be useful when:

  • there are many candidate predictors;
  • some predictors may be weak or redundant;
  • a simpler model is preferred;
  • clinical measurement burden matters;
  • the dataset is too small for ordinary regression with all predictors;
  • the goal is prediction, not causal explanation.

A smaller model may be easier to implement in practice.

4. Lasso and feature selection

Feature selection means choosing a subset of predictors.

Lasso performs feature selection by setting some coefficients to zero.

This can create a sparse model.

A sparse model uses only a limited number of predictors.

However, sparse does not automatically mean good.

A smaller model can still be poorly validated, miscalibrated, or clinically unhelpful.

Scene: selected does not mean causal

Dr. Clinic: Lasso selected biomarker A. Can we say biomarker A causes the disease?

Prof. Stat: No. Lasso selection means the biomarker helped prediction in this dataset.

Learner: It could be correlated with something else?

Prof. Stat: Exactly. It may be a proxy, a correlated marker, or even a dataset-specific signal.

5. Lambda in lasso

Lasso uses a tuning parameter called lambda.

Lambda controls the amount of shrinkage.

Small lambda  → less shrinkage, more predictors kept
Large lambda  → more shrinkage, fewer predictors kept

Different lambda values can produce different selected features.

This is why lambda should be chosen carefully using training data or internal validation.

6. Lasso versus ridge

Ridge and lasso are related, but they behave differently.

Method What it does
Ridge Shrinks coefficients but usually keeps all predictors
Lasso Shrinks some coefficients to zero and selects predictors

Ridge is useful when many predictors may each contain useful information.

Lasso is useful when we want a smaller selected predictor set.

7. Correlated predictors

Lasso can be unstable when predictors are strongly correlated.

Suppose two biomarkers carry very similar information.

Lasso may select biomarker A and remove biomarker B.

In another sample, it may select biomarker B and remove biomarker A.

This does not necessarily mean one biomarker is truly better.

It may simply reflect instability caused by correlation.

8. Biomedical example

Suppose we have:

300 patients
500 candidate biomarkers

Lasso may select 15 biomarkers.

This looks useful.

But before trusting the selected biomarkers, ask:

  • Were they selected inside validation?
  • Are they stable across resamples?
  • Are they biologically plausible?
  • Are they measured reliably?
  • Do they validate externally?
  • Could batch effects explain the selection?

9. Data leakage with lasso

Lasso can easily create leakage.

An incorrect workflow is:

1. Run lasso on the full dataset.
2. Keep selected predictors.
3. Cross-validate a model using those predictors.
4. Report cross-validated AUC.

This is wrong because the validation folds influenced feature selection.

The reported AUC may be too optimistic.

10. Correct workflow

A better workflow is:

For each cross-validation fold:
    1. Use only the training fold.
    2. Run lasso inside the training fold.
    3. Select predictors inside the training fold.
    4. Fit the model inside the training fold.
    5. Evaluate on the validation fold.

The validation fold must remain unseen during feature selection.

This is especially important in biomedical data.

Scene: the leakage monster

Leakage Monster: Select features using the full dataset first. The AUC will look amazing.

Learner: That sounds tempting.

Prof. Stat: And biased. The validation data would already have influenced the model.

Dr. Clinic: So the model may fail on real patients?

Prof. Stat: Exactly.

11. Lasso in clinical prediction

A lasso model should be judged by:

  • discrimination;
  • calibration;
  • Brier score;
  • clinical usefulness;
  • feature stability;
  • external validation;
  • feasibility of predictor measurement.

A lasso model with a neat feature list but poor calibration may still be clinically unsafe.

12. When lasso may be useful

Lasso may be useful when:

  • there are many predictors;
  • a smaller model is desired;
  • some predictors are likely irrelevant;
  • measurement burden matters;
  • the goal is prediction.

Lasso may be less useful when predictors are highly correlated and feature stability is important.

In those settings, elastic net may be more stable.

Common mistake

Do not treat lasso-selected predictors as automatically causal, clinically proven, or biologically validated.

Key takeaway

Lasso regression can shrink coefficients and select predictors, but feature selection must happen inside validation and selected features should be interpreted cautiously.
