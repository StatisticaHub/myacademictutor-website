Exercises

This section contains practice exercises for Machine Learning in Biostatistics.

Exercises are designed to test both conceptual understanding and applied reasoning. Many exercises include answer sections so learners can first attempt the question and then check their reasoning.

How to use exercises

For each topic:

  1. Try the exercise before checking the answer.
  2. Write your answer in your own words.
  3. Compare your answer with the explanation.
  4. Revisit the lecture or note if needed.
  5. Apply the idea in R where possible.
