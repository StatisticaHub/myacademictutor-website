Lectures

This section contains story-based lecture pages for learning machine learning in biostatistics.

The lectures introduce each topic through conversations, common doubts, clinical examples, and modelling mistakes. They are designed to build intuition before you move to the detailed notes, exercises, R code, and case studies.

How to use the lectures

A good learning pathway is:

  1. Read the lecture for intuition.
    Start with the story-based explanation to understand the motivation, common doubts and clinical context.

  2. Read the detailed note.
    Use the notes for deeper statistical theory, equations, model assumptions and interpretation.

  3. Complete the exercise page.
    Test your conceptual understanding before moving to implementation.

  4. Run the R code.
    Use the code section to connect the method to reproducible analysis.

  5. Work through the related case study.
    Apply the idea to a realistic medical prediction problem.

Lecture style

Each lecture follows a similar structure:

  • opening scene;
  • today’s big question;
  • intuition first;
  • mini conversation;
  • medical example;
  • common mistake;
  • clinical reality check;
  • key takeaway;
  • reflection question.

The lectures are designed to be accessible and engaging, while the notes remain more detailed, technical and academic.
