Logistic regression introduction

Mr. R · Prof Stat · ~65 min · Final regression foundation lesson
📍 Scene — regression classroom, when the outcome is yes/no
EM
Emma

Linear regression works when the outcome is numerical. But what if the outcome is yes or no, like disease present or absent?

MR
Mr. R

Then ordinary linear regression is usually not appropriate. For a binary outcome, logistic regression is one of the standard models.

OL
Oliver

Binary means the outcome can only take two values?

MR
Mr. R

Exactly. We often code the outcome as \(Y=1\) for event and \(Y=0\) for no event.

The problem with linear regression for binary outcomes

If \(Y\) is binary, then the conditional mean of \(Y\) is a probability:

\[ E(Y\mid X=x)=P(Y=1\mid X=x) \]

A linear model can predict values below 0 or above 1, which cannot be probabilities. Logistic regression solves this by modelling the probability through a transformation.

Probability, odds and log-odds

If the probability of an event is \(p\), the odds are:

\[ \text{odds}=\frac{p}{1-p} \]

The log-odds, also called the logit, are:

\[ \operatorname{logit}(p)=\log\left(\frac{p}{1-p}\right) \]
JA
James

Why model log-odds instead of probability directly?

MR
Mr. R

Because probabilities are constrained between 0 and 1, while log-odds can range from negative infinity to positive infinity. That makes them suitable for a linear model.

The logistic regression model

With one predictor, logistic regression models:

\[ \log\left(\frac{p_i}{1-p_i}\right)=\beta_0+\beta_1x_i \]

where:

\[ p_i=P(Y_i=1\mid X_i=x_i) \]

Solving for \(p_i\), we get the logistic curve:

\[ p_i=\frac{e^{\beta_0+\beta_1x_i}}{1+e^{\beta_0+\beta_1x_i}} \]

Equivalently:

\[ p_i=\frac{1}{1+e^{-(\beta_0+\beta_1x_i)}} \]

Interpreting the coefficient

The coefficient \(\beta_1\) is a change in log-odds for a one-unit increase in \(X\). That is mathematically correct, but not always easy to interpret.

\[ \beta_1=\text{change in log-odds per one-unit increase in }X \]

Exponentiating the coefficient gives an odds ratio:

\[ \operatorname{OR}=e^{\beta_1} \]
SO
Sophia

If \(e^{\beta_1}=1.5\), does that mean the probability is 1.5 times larger?

MR
Mr. R

No. It means the odds are multiplied by 1.5 for a one-unit increase in \(X\), not the probability. Odds ratios and risk ratios are different.

Multiple logistic regression

With several predictors, the model becomes:

\[ \log\left(\frac{p_i}{1-p_i}\right) = \beta_0+\beta_1X_{1i}+\beta_2X_{2i}+\cdots+\beta_kX_{ki} \]

Each coefficient is an adjusted log-odds coefficient. Exponentiating a coefficient gives an adjusted odds ratio, holding other predictors fixed.

Maximum likelihood, not least squares

Linear regression was fit using least squares. Logistic regression is usually fit using maximum likelihood. The model chooses coefficients that make the observed pattern of zeros and ones most plausible under the model.

\[ L(\boldsymbol{\beta}) = \prod_{i=1}^{n}p_i^{y_i}(1-p_i)^{1-y_i} \]

Rather than minimising squared residuals, logistic regression maximises the likelihood, or equivalently maximises the log-likelihood.

Prediction and classification

Logistic regression predicts probabilities. Classification is a separate decision step. For example, one may classify as positive if the predicted probability is greater than 0.5, but that threshold is not automatic. It depends on the context and the costs of false positives and false negatives.

EM
Emma

So a predicted probability of 0.72 is not itself a category?

MR
Mr. R

Correct. It is an estimated risk or probability. Turning it into a decision requires a threshold and context.

Model assessment basics

For logistic regression, we assess calibration, discrimination, influential observations, model specification and clinical or practical usefulness. Common summaries include predicted probabilities, calibration plots, confusion matrices, sensitivity, specificity and area under the ROC curve.

Common mistake: odds ratio versus probability

An odds ratio of 2 does not mean the probability doubles. It means the odds double. When an event is rare, odds ratios and risk ratios can be similar. When the event is common, they can differ substantially.

Key takeaways from this lecture
  • Logistic regression is used for binary outcomes coded 0/1.
  • The conditional mean of a binary outcome is a probability.
  • Logistic regression models the log-odds as a linear function of predictors.
  • The logistic function keeps predicted probabilities between 0 and 1.
  • A coefficient is a change in log-odds.
  • Exponentiating a coefficient gives an odds ratio.
  • Logistic regression is usually fit by maximum likelihood.
  • Predicted probabilities are not the same as final classifications.
Detailed theoretical notes

Logistic regression introduction

1. Binary outcomes

A binary outcome takes two possible values. We usually code:

\[ Y_i= \begin{cases} 1,& \text{event occurs}\\ 0,& \text{event does not occur} \end{cases} \]

For a binary variable:

\[ E(Y_i\mid X_i)=P(Y_i=1\mid X_i) \]

Thus modelling the conditional mean of \(Y\) is equivalent to modelling the probability of the event.

2. Why not ordinary linear regression?

A linear probability model has the form:

\[ P(Y=1\mid X=x)=\beta_0+\beta_1x \]

This can create predicted probabilities below 0 or above 1. Also, the error variance is not constant for binary outcomes. Logistic regression avoids these problems by transforming the probability scale.

3. Odds

If the probability of an event is \(p\), then the odds are:

\[ \text{odds}=\frac{p}{1-p} \]

For example, if \(p=0.75\), then:

\[ \text{odds}=\frac{0.75}{0.25}=3 \]

This means the event is three times as likely to occur as not occur.

4. Log-odds and the logit link

The logit transformation is:

\[ \operatorname{logit}(p)=\log\left(\frac{p}{1-p}\right) \]

The logit maps probabilities from \((0,1)\) to the whole real line \((-\infty,\infty)\). This allows us to use a linear predictor.

5. Simple logistic regression model

For one predictor \(X_i\), the model is:

\[ \log\left(\frac{p_i}{1-p_i}\right)=\beta_0+\beta_1X_i \]

Let:

\[ \eta_i=\beta_0+\beta_1X_i \]

Then:

\[ \log\left(\frac{p_i}{1-p_i}\right)=\eta_i \]

Exponentiate both sides:

\[ \frac{p_i}{1-p_i}=e^{\eta_i} \]

Solve for \(p_i\):

\[ p_i=e^{\eta_i}(1-p_i) \]
\[ p_i=e^{\eta_i}-e^{\eta_i}p_i \]
\[ p_i(1+e^{\eta_i})=e^{\eta_i} \]
\[ p_i=\frac{e^{\eta_i}}{1+e^{\eta_i}} = \frac{1}{1+e^{-\eta_i}} \]

6. Coefficient interpretation

In the model:

\[ \log\left(\frac{p_i}{1-p_i}\right)=\beta_0+\beta_1X_i \]

a one-unit increase in \(X\) changes the log-odds by \(\beta_1\).

\[ \Delta\operatorname{logit}(p)=\beta_1 \]

On the odds scale:

\[ \frac{\text{odds at }X+1}{\text{odds at }X}=e^{\beta_1} \]

So \(e^{\beta_1}\) is the odds ratio for a one-unit increase in \(X\).

7. Multiple logistic regression

For multiple predictors:

\[ \log\left(\frac{p_i}{1-p_i}\right) = \beta_0+\beta_1X_{1i}+\beta_2X_{2i}+\cdots+\beta_kX_{ki} \]

The coefficient \(\beta_j\) is the adjusted change in log-odds for a one-unit increase in \(X_j\), holding the other variables fixed. The adjusted odds ratio is:

\[ \operatorname{OR}_j=e^{\beta_j} \]

8. Likelihood for binary outcomes

For each observation:

\[ P(Y_i=y_i)=p_i^{y_i}(1-p_i)^{1-y_i} \]

For independent observations, the likelihood is:

\[ L(\boldsymbol{\beta}) = \prod_{i=1}^{n}p_i^{y_i}(1-p_i)^{1-y_i} \]

The log-likelihood is:

\[ \ell(\boldsymbol{\beta}) = \sum_{i=1}^{n} \left[ y_i\log(p_i)+(1-y_i)\log(1-p_i) \right] \]

Maximum likelihood chooses the coefficients that maximise this log-likelihood.

9. Odds ratios versus risk ratios

An odds ratio compares odds:

\[ \operatorname{OR}= \frac{p_1/(1-p_1)}{p_0/(1-p_0)} \]

A risk ratio compares probabilities:

\[ \operatorname{RR}=\frac{p_1}{p_0} \]

These are not the same except approximately when events are rare.

10. Classification threshold

Logistic regression produces predicted probabilities:

\[ \hat{p}_i=P(Y_i=1\mid X_i) \]

Classification requires choosing a threshold \(c\):

\[ \hat{Y}_i= \begin{cases} 1,& \hat{p}_i\geq c\\ 0,& \hat{p}_i

The threshold should reflect the purpose of the model and the relative costs of false positives and false negatives.

11. Model assessment concepts

ConceptMeaning
CalibrationWhether predicted probabilities agree with observed event frequencies
DiscriminationWhether the model assigns higher probabilities to events than non-events
SensitivityProportion of true events correctly classified positive
SpecificityProportion of true non-events correctly classified negative
Decision thresholdProbability cut-off used for classification

12. Common mistakes

  • Interpreting odds ratios as probability ratios: odds and probabilities are different.
  • Using linear regression without checking probability limits: predicted values may fall outside 0 and 1.
  • Assuming 0.5 is always the best threshold: thresholds depend on context.
  • Thinking classification is the same as probability estimation: classification is a decision step after prediction.
  • Ignoring calibration: a model can rank people well but give poorly calibrated probabilities.
  • Claiming causality from an odds ratio alone: causal interpretation still needs study design and assumptions.
Notes summary
  • Binary outcomes are often coded as 0/1.
  • Logistic regression models log-odds as a linear function of predictors.
  • The logistic function maps the linear predictor to a probability.
  • Coefficients are log-odds changes; exponentiated coefficients are odds ratios.
  • Odds ratios are not the same as risk ratios.
  • Logistic regression is fit using maximum likelihood.
  • Model assessment includes calibration, discrimination and threshold-based performance.

Practice exercises

Logistic regression introduction

Exercise 1
Convert probability to odds
Easy+

If \(p=0.80\), calculate the odds.

Answer

\(\text{odds}=p/(1-p)=0.80/0.20=4\). The event is four times as likely as the non-event.

Exercise 2
Convert odds to probability
Easy+

If the odds are 3, calculate the probability.

Answer

If odds \(=p/(1-p)=3\), then \(p=\text{odds}/(1+\text{odds})=3/(1+3)=0.75\).

Exercise 3
Interpret a coefficient
Medium+

In a logistic regression model, \(\beta_1=0.40\). Interpret this on the log-odds scale.

Answer

For a one-unit increase in \(X\), the log-odds of the event increase by 0.40, holding other variables fixed if present.

Exercise 4
Calculate an odds ratio
Medium+

If \(\beta_1=0.40\), calculate the odds ratio approximately.

Answer

\(\operatorname{OR}=e^{0.40}\approx1.49\). The odds are multiplied by about 1.49 for each one-unit increase in \(X\).

Exercise 5
Compute probability from linear predictor
Medium+

If \(\eta=0\), calculate \(p=1/(1+e^{-\eta})\).

Answer

If \(\eta=0\), then \(p=1/(1+e^0)=1/(1+1)=0.5\).

Exercise 6
Odds ratio versus risk ratio
Hard+

Why is it wrong to say that an odds ratio of 2 always means the probability doubles?

Answer

An odds ratio compares odds, not probabilities. When the outcome is common, doubling the odds can correspond to much less than doubling the probability.

Exercise 7
Classification threshold
Hard+

A patient has predicted probability 0.42. Does this automatically mean they are classified as negative?

Answer

Not automatically. Classification depends on the chosen threshold. If the threshold is 0.5, they would be classified negative; if the threshold is 0.3, they would be classified positive.

Exercise 8
Maximum likelihood
Hard+

Explain in words how logistic regression is usually fitted.

Answer

Logistic regression is usually fitted by maximum likelihood. The coefficients are chosen to make the observed pattern of 0s and 1s as plausible as possible under the model.
