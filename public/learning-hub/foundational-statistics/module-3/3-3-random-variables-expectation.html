Random variables and expectation

Mr. R · Prof Stat · ~40 min · After conditional probability
📍 Scene — probability classroom, moving from events to numerical variables
EM
Emma

So far, probability has been about events like \(A\), \(B\), disease, positive tests, and overlaps. But statistics usually works with numbers. Where do numbers enter probability?

MR
Mr. R

Excellent question. Numbers enter through random variables. A random variable converts uncertain outcomes into numerical values.

OL
Oliver

So a random variable is not just a variable in a spreadsheet?

MR
Mr. R

Not exactly. A spreadsheet variable is observed data. A random variable is a mathematical object before observation. It describes a numerical quantity whose value is uncertain.

Why random variables are needed

In probability, the original outcome may not be numerical. For example, when three coins are tossed, the outcome might be \(HHT\), \(THT\), or \(TTT\). But a statistician may care about the number of heads.

So we define a random variable \(X\) as the number of heads. Then \(X(HHT)=2\), \(X(THT)=1\), and \(X(TTT)=0\).

JA
James

So the random variable takes the messy outcome and turns it into a number we can analyse?

MR
Mr. R

Exactly. A random variable is a function from outcomes to numbers.

SO
Sophia

Then before we toss the coins, \(X\) is uncertain. After we toss them, \(X\) has an observed value?

MR
Mr. R

Perfect. That is the difference between a random variable and its realised value.

Discrete and continuous random variables

Some random variables count things. Others measure quantities on a continuum. This gives us the distinction between discrete and continuous random variables.

TypeMeaningExamples
DiscreteCountable possible valuesNumber of heads, number of patients responding, number of hospital admissions
ContinuousValues on a continuumHeight, blood pressure, survival time, concentration
EM
Emma

If a variable is continuous, can it take infinitely many possible values?

MR
Mr. R

Yes. In theory, a continuous variable can take any value in an interval. That is why continuous probability is handled differently later using densities and areas.

Probability distribution

A random variable is not fully described by listing possible values. We also need the probability attached to those values. This is called the probability distribution.

OL
Oliver

For a fair die, the random variable \(X\) could be the number showing on the die, and each value 1 to 6 has probability \(1/6\)?

MR
Mr. R

Correct. That table of values and probabilities is the distribution of \(X\).

Expectation

The expected value of a random variable is its probability-weighted average. It is the theoretical mean of the distribution.

\[ E(X)=\sum_x x\,P(X=x) \]
JA
James

Is expected value the same as the value I expect to see next?

MR
Mr. R

Not necessarily. Expected value is a long-run average, not a guaranteed next outcome.

SO
Sophia

Like a fair die has expected value 3.5, but it can never actually show 3.5.

MR
Mr. R

Exactly. That example is one of the best ways to understand expectation.

Variance

Expectation tells us the theoretical centre. Variance tells us the theoretical spread. It measures the expected squared distance from the mean.

\[ \operatorname{Var}(X)=E\left[(X-\mu)^2\right],\qquad \mu=E(X) \]

A useful computational formula is:

\[ \operatorname{Var}(X)=E(X^2)-[E(X)]^2 \]
EM
Emma

Why do we square the distance from the mean?

MR
Mr. R

Squaring makes all distances non-negative and gives larger weight to values far from the mean. It is the theoretical version of variance from descriptive statistics.

Linearity of expectation

One of the most important rules in probability is that expectation is linear.

\[ E(aX+b)=aE(X)+b \]
\[ E(X+Y)=E(X)+E(Y) \]

The second result is especially powerful because it does not require \(X\) and \(Y\) to be independent.

OL
Oliver

So even if two variables are related, the expectation of their sum is still the sum of their expectations?

MR
Mr. R

Yes. Independence is not needed for linearity of expectation. That makes it extremely useful.

Indicator variables

An indicator variable equals 1 if an event occurs and 0 otherwise. If \(I_A\) is the indicator of event \(A\), then:

\[ I_A= \begin{cases} 1, & \text{if } A \text{ occurs},\\ 0, & \text{if } A \text{ does not occur}. \end{cases} \]

Its expectation is:

\[ E(I_A)=P(A) \]
JA
James

That is neat. A probability can be written as an expected value?

MR
Mr. R

Yes, and that idea becomes very powerful later in statistical modelling.

Key takeaways from this lecture
  • A random variable maps uncertain outcomes to numerical values.
  • Discrete random variables have countable possible values.
  • Continuous random variables can take values on a continuum.
  • A probability distribution assigns probabilities to possible values.
  • Expectation is a probability-weighted long-run average.
  • Variance measures theoretical spread around the expectation.
  • Linearity of expectation does not require independence.
  • Indicator variables connect probabilities directly to expected values.
Detailed theoretical notes

Random variables and expectation

1. Why random variables are introduced

Probability begins with outcomes and events. However, statistical analysis usually requires numerical quantities: counts, measurements, scores, means, proportions, and test statistics. Random variables provide the bridge between uncertain outcomes and numerical analysis.

Definition

A random variable is a rule or function that assigns a numerical value to each outcome of a random experiment.

2. Random variable as a function

Let \(S\) be a sample space. A random variable \(X\) is a function from \(S\) to the real numbers:

\[ X:S\to \mathbb{R} \]

This means that for every outcome \(s\in S\), the random variable assigns a number \(X(s)\).

Example: three coin tosses

Suppose three coins are tossed. The sample space is:

\[ S=\{HHH,HHT,HTH,THH,HTT,THT,TTH,TTT\} \]

Define \(X\) as the number of heads. Then:

\[ X(HHH)=3,\quad X(HHT)=2,\quad X(TTT)=0 \]

The random variable \(X\) can take values \(0,1,2,3\).

3. Random variable versus observed value

Before the experiment is performed, \(X\) is uncertain. After the experiment is performed, we observe a realised value, often written as \(x\).

ConceptMeaningExample
Random variable \(X\)Uncertain numerical quantity before observationNumber of heads before tossing coins
Realised value \(x\)Observed numerical value after the experimentThe value \(x=2\) after observing HHT

4. Discrete and continuous random variables

A random variable is discrete if it has countable possible values. A random variable is continuous if it can take values across intervals.

TypePossible valuesExamples
DiscreteFinite or countably infiniteNumber of patients responding, number of errors, number of heads
ContinuousUncountably many values in intervalsHeight, weight, blood pressure, survival time

5. Probability mass function for discrete random variables

For a discrete random variable \(X\), the probability mass function gives the probability that \(X\) takes each possible value.

\[ p(x)=P(X=x) \]

A valid probability mass function must satisfy two conditions:

\[ p(x)\geq 0 \quad \text{for all }x \]
\[ \sum_x p(x)=1 \]

6. Expectation of a discrete random variable

The expected value of a discrete random variable is the probability-weighted average of its possible values.

\[ E(X)=\sum_x x\,P(X=x) \]

The summation is taken over all possible values of \(X\).

Interpretation

The expected value is the long-run average value of the random variable over many repetitions of the random experiment.

7. Worked example: fair die

Let \(X\) be the number shown on a fair die. Then \(X\in\{1,2,3,4,5,6\}\) and each value has probability \(1/6\).

\[ E(X)=1\left(\frac{1}{6}\right)+2\left(\frac{1}{6}\right)+3\left(\frac{1}{6}\right)+4\left(\frac{1}{6}\right)+5\left(\frac{1}{6}\right)+6\left(\frac{1}{6}\right) \]
\[ E(X)=\frac{1+2+3+4+5+6}{6}=\frac{21}{6}=3.5 \]

The expected value is \(3.5\), although no single die roll can equal 3.5. This shows that expectation is a long-run average, not necessarily a possible outcome.

8. Expectation of a transformed random variable

If \(g(X)\) is a function of \(X\), then:

\[ E[g(X)]=\sum_x g(x)P(X=x) \]

For example:

\[ E(X^2)=\sum_x x^2P(X=x) \]

9. Variance

Let \(\mu=E(X)\). The variance of \(X\) is the expected squared distance of \(X\) from its mean:

\[ \operatorname{Var}(X)=E\left[(X-\mu)^2\right] \]

Expanding the square gives a useful computational formula.

10. Derivation of the variance shortcut

Start from the definition:

\[ \operatorname{Var}(X)=E\left[(X-\mu)^2\right] \]

Expand the squared term:

\[ (X-\mu)^2=X^2-2\mu X+\mu^2 \]

Take expectation on both sides:

\[ E\left[(X-\mu)^2\right]=E(X^2)-2\mu E(X)+E(\mu^2) \]

Since \(\mu\) is a constant, \(E(\mu^2)=\mu^2\), and since \(E(X)=\mu\):

\[ \operatorname{Var}(X)=E(X^2)-2\mu^2+\mu^2 \]

Therefore:

\[ \operatorname{Var}(X)=E(X^2)-\mu^2 \]

Since \(\mu=E(X)\), this is:

\[ \operatorname{Var}(X)=E(X^2)-[E(X)]^2 \]

11. Standard deviation

The standard deviation of a random variable is the square root of its variance:

\[ \operatorname{SD}(X)=\sqrt{\operatorname{Var}(X)} \]

Variance is measured in squared units, while standard deviation returns to the original units.

12. Linearity of expectation

For constants \(a\) and \(b\):

\[ E(aX+b)=aE(X)+b \]

Derivation

\[ E(aX+b)=\sum_x (ax+b)P(X=x) \]
\[ E(aX+b)=a\sum_x xP(X=x)+b\sum_x P(X=x) \]

Since \(\sum_x P(X=x)=1\), we obtain:

\[ E(aX+b)=aE(X)+b \]

13. Expectation of a sum

For two random variables \(X\) and \(Y\):

\[ E(X+Y)=E(X)+E(Y) \]

This result does not require independence. It holds generally whenever the expectations exist.

14. Variance under scaling and shifting

If \(a\) and \(b\) are constants, then:

\[ \operatorname{Var}(aX+b)=a^2\operatorname{Var}(X) \]

Adding a constant shifts the random variable but does not change its spread. Multiplying by \(a\) scales the spread by \(a^2\) in variance.

15. Bernoulli random variable

A Bernoulli random variable takes value 1 with probability \(p\), and 0 with probability \(1-p\).

\[ X\sim \operatorname{Bernoulli}(p) \]
xP(X=x)
1\(p\)
0\(1-p\)

Expectation of Bernoulli

\[ E(X)=1(p)+0(1-p)=p \]

Variance of Bernoulli

First calculate \(E(X^2)\). Since \(X\) is either 0 or 1, \(X^2=X\). Therefore:

\[ E(X^2)=E(X)=p \]

Using the variance shortcut:

\[ \operatorname{Var}(X)=E(X^2)-[E(X)]^2=p-p^2=p(1-p) \]

16. Indicator variables

An indicator variable for event \(A\), written \(I_A\), is defined by:

\[ I_A= \begin{cases} 1, & \text{if } A \text{ occurs},\\ 0, & \text{if } A \text{ does not occur}. \end{cases} \]

Since \(I_A\) is Bernoulli with success probability \(P(A)\):

\[ E(I_A)=P(A) \]

This result is important because it connects probability to expectation.

17. Common mistakes

  • Thinking a random variable is random after observation: after observation, it has a realised value.
  • Thinking expected value must be observable: a die has expected value 3.5, but cannot show 3.5.
  • Forgetting probabilities in expectation: expectation is weighted by probabilities.
  • Confusing variance and standard deviation: standard deviation is the square root of variance.
  • Assuming \(E(XY)=E(X)E(Y)\) always: this generally requires independence, unlike \(E(X+Y)=E(X)+E(Y)\).
Notes summary
  • A random variable maps outcomes to numbers.
  • A probability mass function assigns probabilities to discrete values.
  • Expectation is a probability-weighted theoretical mean.
  • Variance is expected squared deviation from the mean.
  • The shortcut \(\operatorname{Var}(X)=E(X^2)-[E(X)]^2\) follows by expanding \((X-\mu)^2\).
  • Linearity of expectation is valid without independence.
  • Bernoulli and indicator variables are fundamental building blocks.

Practice exercises

Random variables and expectation

Try each exercise before revealing the answer. The focus is on definitions, expectation, variance and interpretation.

Exercise 1
Define a random variable
Easy+

Two coins are tossed. Define \(X\) as the number of heads. List the possible values of \(X\).

Answer

The sample outcomes are \(HH,HT,TH,TT\). The possible values of \(X\) are \(0,1,2\).

Exercise 2
Check a probability mass function
Easy+

A random variable has probabilities \(0.2\), \(0.5\), and \(0.3\) for three possible values. Is this a valid probability distribution?

Answer

Yes. All probabilities are non-negative and they sum to \(0.2+0.5+0.3=1\).

Exercise 3
Expected value
Medium+

Let \(X\) take values 0, 1 and 2 with probabilities 0.2, 0.5 and 0.3. Calculate \(E(X)\).

Answer

\(E(X)=0(0.2)+1(0.5)+2(0.3)=0+0.5+0.6=1.1\).

Exercise 4
Variance shortcut
Medium+

For the same \(X\), calculate \(\operatorname{Var}(X)\).

Answer

First, \(E(X^2)=0^2(0.2)+1^2(0.5)+2^2(0.3)=0+0.5+1.2=1.7\). Since \(E(X)=1.1\), \(\operatorname{Var}(X)=E(X^2)-[E(X)]^2=1.7-1.21=0.49\).

Exercise 5
Bernoulli expectation and variance
Medium+

If \(X\sim\operatorname{Bernoulli}(0.7)\), find \(E(X)\) and \(\operatorname{Var}(X)\).

Answer

For Bernoulli \(p\), \(E(X)=p\) and \(\operatorname{Var}(X)=p(1-p)\). Therefore, \(E(X)=0.7\) and \(\operatorname{Var}(X)=0.7(0.3)=0.21\).

Exercise 6
Linear transformation
Hard+

Suppose \(E(X)=4\) and \(\operatorname{Var}(X)=9\). Let \(Y=3X-2\). Find \(E(Y)\) and \(\operatorname{Var}(Y)\).

Answer

\(E(Y)=E(3X-2)=3E(X)-2=3(4)-2=10\). Also, \(\operatorname{Var}(Y)=\operatorname{Var}(3X-2)=3^2\operatorname{Var}(X)=9(9)=81\).

Exercise 7
Indicator variable
Hard+

Let \(I_A\) be the indicator variable for event \(A\). If \(P(A)=0.35\), find \(E(I_A)\) and explain why.

Answer

The indicator \(I_A\) equals 1 if \(A\) occurs and 0 otherwise. Therefore, \(E(I_A)=1P(A)+0P(A^c)=P(A)=0.35\).

Exercise 8
Interpret expectation
Hard+

Explain why the expected value of a die roll is \(3.5\), even though 3.5 is not a possible die outcome.

Answer

The expected value is a long-run average, not a guaranteed or necessarily observable value. Across many rolls of a fair die, the average outcome tends toward \(3.5\), although no individual roll can equal \(3.5\).
