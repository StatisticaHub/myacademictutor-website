Discrete distributions

Mr. R · Prof Stat · ~42 min · After random variables and expectation
📍 Scene — probability classroom, named models on the board
EM
Emma

Last lesson we learned random variables and expectation. Now I keep seeing named distributions: Bernoulli, binomial, geometric, Poisson. Why do we need so many names?

MR
Mr. R

Because common random processes appear again and again. A named distribution is a reusable probability model for a specific kind of data-generating process.

OL
Oliver

So we should not just memorise formulas. We should ask what process produced the data?

MR
Mr. R

Exactly. Distribution choice is not decoration. It is a modelling decision.

What is a discrete distribution?

A discrete distribution describes a random variable whose possible values are countable. The values may be finite, such as 0 or 1, or countably infinite, such as 0, 1, 2, 3 and so on.

For a discrete random variable \(X\), the distribution tells us the value of \(P(X=x)\) for each possible value \(x\).

JA
James

So a distribution is not just the list of values. It is the list of values and probabilities?

MR
Mr. R

Correct. A distribution is a complete probability description of the random variable.

Bernoulli distribution

The simplest discrete distribution is the Bernoulli distribution. It models one binary trial: success or failure, yes or no, response or no response, infected or not infected.

\[ X\sim \operatorname{Bernoulli}(p) \]

The random variable takes value 1 with probability \(p\), and 0 with probability \(1-p\).

SO
Sophia

So if \(X=1\) means a patient responds to treatment, then \(p\) is the probability of response?

MR
Mr. R

Exactly. Bernoulli is the probability model for a single binary outcome.

Binomial distribution

The binomial distribution extends Bernoulli. Instead of one binary trial, it counts the number of successes in a fixed number of independent Bernoulli trials with the same success probability.

\[ X\sim \operatorname{Binomial}(n,p) \]

Here, \(n\) is the number of trials and \(p\) is the success probability on each trial.

EM
Emma

For example, if 20 patients are treated and each either responds or does not respond, \(X\) could be the number who respond?

MR
Mr. R

Yes, if we are willing to assume a fixed number of patients, independent responses, and the same response probability for each patient.

OL
Oliver

So the assumptions are part of the distribution?

MR
Mr. R

Absolutely. A formula without assumptions is dangerous.

Where the binomial formula comes from

If exactly \(x\) successes occur in \(n\) trials, one particular arrangement has probability \(p^x(1-p)^{n-x}\). But the \(x\) successes can occur in many different positions. The number of arrangements is \(\binom{n}{x}\).

\[ P(X=x)=\binom{n}{x}p^x(1-p)^{n-x} \]
JA
James

So the binomial coefficient counts the number of ways to arrange the successes?

MR
Mr. R

Exactly. The probability part gives the chance of one arrangement; the coefficient counts how many equivalent arrangements exist.

Geometric distribution

The geometric distribution answers a different question: how many trials are needed until the first success?

\[ X\sim \operatorname{Geometric}(p) \]

If \(X\) is the trial number of the first success, then:

\[ P(X=k)=(1-p)^{k-1}p,\qquad k=1,2,3,\ldots \]
SO
Sophia

So for the first success on trial 5, the first 4 trials must fail, then trial 5 succeeds?

MR
Mr. R

Exactly. That gives \((1-p)^4p\).

Poisson distribution

The Poisson distribution models counts of events in a fixed interval of time, space, area, volume, or exposure, when events occur at an average rate.

\[ X\sim \operatorname{Poisson}(\lambda) \]

The parameter \(\lambda\) is the expected count in the interval.

\[ P(X=x)=\frac{e^{-\lambda}\lambda^x}{x!},\qquad x=0,1,2,\ldots \]
EM
Emma

What kinds of things are Poisson counts?

MR
Mr. R

Examples include calls per hour, disease cases per week, mutations in a genomic region, hospital arrivals per shift, or defects per manufactured batch.

OL
Oliver

So Poisson is not for fixed number of trials like binomial?

MR
Mr. R

Correct. Binomial counts successes out of fixed trials. Poisson counts events over exposure.

The modelling question

The most important skill is not memorising the formula. It is matching the random process to the correct distribution.

Key takeaways from this lecture
  • A discrete distribution assigns probabilities to countable values.
  • Bernoulli models one binary outcome.
  • Binomial counts successes in a fixed number of independent Bernoulli trials.
  • Geometric models the waiting time until the first success.
  • Poisson models event counts over a fixed interval or exposure.
  • Distribution choice must follow the data-generating process and assumptions.
Detailed theoretical notes

Discrete distributions

1. What is a discrete probability distribution?

A discrete probability distribution describes a random variable that can take countable values. The distribution assigns a probability to each possible value.

\[ p(x)=P(X=x) \]

For a valid probability mass function:

\[ p(x)\geq 0\quad \text{for all }x \]
\[ \sum_x p(x)=1 \]

2. Bernoulli distribution

A Bernoulli random variable models one binary trial. It takes value 1 for success and 0 for failure.

\[ X\sim \operatorname{Bernoulli}(p) \]
xP(X=x)
1\(p\)
0\(1-p\)

Expectation

\[ E(X)=1(p)+0(1-p)=p \]

Variance

Since \(X^2=X\) for a Bernoulli variable:

\[ E(X^2)=E(X)=p \]
\[ \operatorname{Var}(X)=E(X^2)-[E(X)]^2=p-p^2=p(1-p) \]

3. Binomial distribution

A binomial random variable counts the number of successes in \(n\) Bernoulli trials.

\[ X\sim \operatorname{Binomial}(n,p) \]

The assumptions are:

  • There is a fixed number of trials \(n\).
  • Each trial has two outcomes: success or failure.
  • The trials are independent.
  • The success probability \(p\) is the same for each trial.
  • The random variable counts the number of successes.

Derivation of the binomial probability formula

Suppose exactly \(x\) successes occur in \(n\) trials. One specific arrangement has probability:

\[ p^x(1-p)^{n-x} \]

There are \(\binom{n}{x}\) ways to choose which \(x\) of the \(n\) trials are successes:

\[ \binom{n}{x}=\frac{n!}{x!(n-x)!} \]

Therefore:

\[ P(X=x)=\binom{n}{x}p^x(1-p)^{n-x},\qquad x=0,1,\ldots,n \]

Mean and variance of binomial

A binomial random variable can be written as the sum of \(n\) Bernoulli random variables:

\[ X=X_1+X_2+\cdots+X_n \]

where each \(X_i\sim\operatorname{Bernoulli}(p)\).

Using linearity of expectation:

\[ E(X)=E(X_1)+E(X_2)+\cdots+E(X_n)=p+p+\cdots+p=np \]

If the trials are independent, variances add:

\[ \operatorname{Var}(X)=np(1-p) \]

4. Geometric distribution

A geometric random variable models the trial number on which the first success occurs.

\[ X\sim \operatorname{Geometric}(p) \]

For the first success to occur on trial \(k\), the first \(k-1\) trials must be failures and the \(k\)-th trial must be a success:

\[ P(X=k)=(1-p)^{k-1}p,\qquad k=1,2,3,\ldots \]

The mean and variance are:

\[ E(X)=\frac{1}{p} \]
\[ \operatorname{Var}(X)=\frac{1-p}{p^2} \]
Convention warning

Some textbooks define the geometric distribution as the number of failures before the first success. In this course, \(X\) is the trial number of the first success.

5. Poisson distribution

A Poisson random variable models the number of events occurring in a fixed interval of time, area, volume, or exposure.

\[ X\sim \operatorname{Poisson}(\lambda) \]

The probability mass function is:

\[ P(X=x)=\frac{e^{-\lambda}\lambda^x}{x!},\qquad x=0,1,2,\ldots \]

The mean and variance are both equal to \(\lambda\):

\[ E(X)=\lambda \]
\[ \operatorname{Var}(X)=\lambda \]

6. When is Poisson reasonable?

A Poisson model is often considered when:

  • the outcome is a count,
  • events occur over a fixed exposure interval,
  • events are approximately independent,
  • the event rate is approximately constant within the interval,
  • two events are unlikely to occur at exactly the same instant or location.

7. Overdispersion

For a Poisson distribution, the mean and variance are equal. In real count data, the variance is often larger than the mean because of clustering, unmeasured heterogeneity, or unequal exposure.

Overdispersion

Overdispersion occurs when observed count data have variance larger than the mean. In that case, a simple Poisson model may underestimate uncertainty.

9. Common mistakes

  • Using binomial without fixed \(n\): binomial requires a fixed number of trials.
  • Ignoring unequal probabilities: binomial assumes common success probability \(p\).
  • Ignoring dependence: binomial variance formula requires independent trials.
  • Using Poisson for bounded counts: Poisson is not ideal when there is a clear fixed maximum number of successes.
  • Ignoring exposure: count data should often be interpreted relative to time, population size, area, or another exposure measure.
  • Confusing geometric conventions: check whether \(X\) counts trial number or failures before success.
Notes summary
  • Discrete distributions model countable random variables.
  • Bernoulli is the model for one binary trial.
  • Binomial is the model for the number of successes in fixed independent trials.
  • Geometric is the model for waiting until the first success.
  • Poisson is the model for counts over an interval or exposure.
  • The assumptions matter as much as the formulas.

Practice exercises

Discrete distributions

Try each exercise before revealing the answer. Focus on matching the distribution to the data-generating process.

Exercise 1
Bernoulli mean and variance
Easy+

If \(X\sim\operatorname{Bernoulli}(0.8)\), find \(E(X)\) and \(\operatorname{Var}(X)\).

Answer

\(E(X)=p=0.8\). Also, \(\operatorname{Var}(X)=p(1-p)=0.8(0.2)=0.16\).

Exercise 2
Binomial model conditions
Easy+

List four conditions required for a binomial model.

Answer

There must be a fixed number of trials, each trial must be binary, trials should be independent, and the success probability \(p\) should be the same on each trial.

Exercise 3
Binomial probability
Medium+

Let \(X\sim\operatorname{Binomial}(5,0.4)\). Calculate \(P(X=2)\).

Answer

\(P(X=2)=\binom{5}{2}(0.4)^2(0.6)^3=10(0.16)(0.216)=0.3456\).

Exercise 4
Binomial mean and variance
Medium+

If \(X\sim\operatorname{Binomial}(40,0.25)\), find \(E(X)\) and \(\operatorname{Var}(X)\).

Answer

\(E(X)=np=40(0.25)=10\). \(\operatorname{Var}(X)=np(1-p)=40(0.25)(0.75)=7.5\).

Exercise 5
Geometric probability
Medium+

Let \(X\sim\operatorname{Geometric}(0.2)\), where \(X\) is the trial number of the first success. Calculate \(P(X=4)\).

Answer

\(P(X=4)=(1-0.2)^{3}(0.2)=0.8^3(0.2)=0.512(0.2)=0.1024\).

Exercise 6
Poisson probability
Hard+

Let \(X\sim\operatorname{Poisson}(3)\). Calculate \(P(X=2)\).

Answer

\(P(X=2)=\frac{e^{-3}3^2}{2!}=\frac{9e^{-3}}{2}\approx0.2240\).

Exercise 7
Choose the distribution
Hard+

A hospital records the number of emergency arrivals per hour. Which distribution might be considered first, and what assumptions should be checked?

Answer

A Poisson distribution might be considered because the outcome is a count over a fixed time interval. We should check whether arrivals are approximately independent, whether the rate is reasonably constant, and whether the variance is close to the mean.

Exercise 8
Overdispersion
Hard+

In a count dataset, the sample mean is 4.2 and the sample variance is 15.8. Explain why a simple Poisson model may be questionable.

Answer

A Poisson distribution has equal mean and variance. Here the variance is much larger than the mean, suggesting overdispersion. A simple Poisson model may underestimate uncertainty.
