Tables and graphs

Mr. R · Emma, Oliver, James, Sophia · ~30 min · Descriptive presentation
📍 Scene — the class has data, but the raw spreadsheet is hard to read
EM
Emma

We have rows and columns now, but looking at the spreadsheet directly feels confusing. How do we make sense of it?

MR
Mr. R

We organise data using tables and graphs. They help us see patterns, compare groups and communicate findings clearly.

OL
Oliver

So graphs are just for making results look nice?

MR
Mr. R

No. A good graph is a statistical argument. It should reveal structure without misleading the reader.

Big idea

Tables organise data precisely. Graphs reveal patterns visually. Both should match the variable type and the research question.

Why tables and graphs matter

Before formal analysis, tables and graphs help us understand the basic structure of a dataset. They show frequencies, distributions, comparisons, relationships and unusual values. They also help detect data errors.

JA
James

Can a graph be mathematically correct but still misleading?

MR
Mr. R

Yes. Poor axis scales, missing labels, 3D effects or inappropriate graph choices can distort interpretation.

Frequency tables

A frequency table shows how often each value or category appears. It is especially useful for categorical data and discrete numerical data.

Bar charts

A bar chart displays categories using separate bars. It is suitable for categorical variables. The bars should have gaps because the categories are distinct.

Histograms

A histogram displays the distribution of a numerical variable by grouping values into intervals called bins. Unlike bar charts, histogram bars touch because the scale is continuous or ordered numerically.

SO
Sophia

So a histogram is not just a bar chart?

MR
Mr. R

Correct. Bar charts show categories. Histograms show distributions of numerical variables.

Line graphs and scatter plots

A line graph is useful when values are measured over time or another ordered sequence. A scatter plot is useful when we want to study the relationship between two numerical variables.

Responsible presentation

Every table or graph should have a clear title, labelled axes, sensible units and an honest scale. The aim is not decoration but accurate communication.

Key takeaways from this lecture
  • Tables provide precise organisation of data.
  • Graphs help reveal visual patterns and comparisons.
  • Bar charts are for categorical variables.
  • Histograms are for numerical distributions.
  • Line graphs are useful for time trends.
  • Scatter plots show relationships between two numerical variables.
  • Good visualisation should be clear, honest and matched to the data type.
Detailed notes

Tables and graphs

1. Purpose of data display

Tables and graphs are tools for statistical communication. They help us organise data, inspect patterns and present evidence. They should be designed around the research question and the variable type.

Definition — data display

A data display is an organised visual or tabular representation of data designed to make structure, comparison or distribution easier to understand.

2. Frequency tables

A frequency table lists each category or value and records how often it occurs.

Relative frequency = frequency ÷ total number of observations

If a category appears \(f\) times in a sample of size \(n\), then its relative frequency is:

Relative frequency = f / n

Percentages are obtained by multiplying relative frequency by 100.

Percentage = (f / n) × 100%

3. Two-way tables

A two-way table summarises two categorical variables at the same time. It helps compare distributions across groups.

PassFailTotal
Attended revision42850
Did not attend302050
Total7228100

Row percentages compare outcomes within rows. Column percentages compare group composition within columns. The correct percentage depends on the question being asked.

4. Bar charts

A bar chart is used for categorical data. Each bar represents a category. Bar height represents frequency, relative frequency or percentage.

Good practice

Use a clear title, label categories, start the frequency axis at zero and avoid unnecessary 3D effects.

5. Pie charts

A pie chart shows parts of a whole. It can be useful for a small number of categories, but it is often harder to compare angles than bar lengths. For many categories, a bar chart is usually clearer.

6. Histograms

A histogram displays the distribution of a numerical variable. The horizontal axis is divided into intervals, and each bar shows how many observations fall into that interval.

Important distinction

Bar charts are for categories. Histograms are for numerical intervals. Histogram bars usually touch because the intervals are adjacent.

7. Choosing bin width in histograms

The number and width of bins affects the appearance of a histogram. Too few bins can hide structure. Too many bins can make random noise look like pattern.

8. Boxplots

A boxplot summarises a numerical variable using the median, quartiles and possible outliers. It is useful for comparing distributions across groups.

IQR = Q₃ − Q₁

The interquartile range measures the spread of the middle 50% of the data.

9. Line graphs

A line graph is useful when observations are ordered, especially over time. The line connects values in sequence, making trends and changes visible.

10. Scatter plots

A scatter plot displays pairs of numerical values. Each point represents one observational unit. Scatter plots help identify direction, strength, shape and unusual observations in relationships.

11. Choosing the correct display

Data type or questionRecommended displayAvoid
One nominal categorical variableFrequency table, bar chartHistogram
One continuous numerical variableHistogram, boxplotPie chart
Two categorical variablesTwo-way table, grouped bar chartScatter plot
Two numerical variablesScatter plotPie chart
Time seriesLine graphUnordered bar chart

12. Common mistakes

  • Using a histogram for categorical labels.
  • Using a bar chart when a histogram is needed.
  • Leaving axes unlabelled.
  • Using misleading scales or truncated axes without explanation.
  • Using too many colours or decorative effects.
  • Choosing percentages without clarifying the denominator.
  • Using a graph that does not answer the research question.

Practice exercises

Tables and graphs

Exercise 1
Relative frequency
Easy+

A category appears 25 times in a sample of 100. Calculate its relative frequency and percentage.

Answer

Relative frequency = 25 / 100 = 0.25. Percentage = 25%.

Exercise 2
Choosing a bar chart
Easy+

Which graph is suitable for showing counts of blood groups A, B, AB and O?

Answer

A bar chart is suitable because blood group is categorical.

Exercise 3
Choosing a histogram
Medium+

Which graph is suitable for showing the distribution of heights in a class?

Answer

A histogram or boxplot is suitable because height is a numerical variable.

Exercise 4
Two numerical variables
Medium+

Which graph would you use to explore the relationship between study hours and exam score?

Answer

A scatter plot is appropriate because both variables are numerical and the goal is to examine their relationship.

Exercise 5
Misleading graph
Hard+

Why can a bar chart with a truncated vertical axis be misleading?

Answer

It can exaggerate differences between categories because the visual length of bars no longer reflects the full scale from zero. If a truncated axis is used, it must be clearly justified and labelled.
