Module 4 · Lesson 4.2

Confidence intervals

Mr. R · Prof Stat · ~48 min · After sampling distributions and standard error
📍 Scene — inference classroom, one estimate on the board
EM
Emma

Last lesson we learned that a sample estimate varies from sample to sample. So if I report only one number, like a sample mean, I am hiding uncertainty?

MR
Mr. R

Exactly. A point estimate is useful, but incomplete. A confidence interval adds a range of plausible values for the population parameter.

OL
Oliver

So a confidence interval is like saying, “The estimate is this, but allow for sampling uncertainty”?

MR
Mr. R

That is a very good first interpretation. It is an estimate plus a margin of error.

From point estimate to interval estimate

Suppose a sample of 100 patients has mean systolic blood pressure \(\bar{x}=128\) mmHg. If the estimated standard error is 2 mmHg, then a rough 95% confidence interval is:

\[ 128 \pm 1.96(2) \]

That gives:

\[ 128 \pm 3.92=(124.08,\;131.92) \]
JA
James

Why multiply the standard error by 1.96?

MR
Mr. R

Because for a standard normal distribution, about 95% of the area lies between \(-1.96\) and \(1.96\). That number is the critical value for a two-sided 95% confidence interval.

The general structure

Most confidence intervals follow the same structure:

\[ \text{estimate}\;\pm\;\text{critical value}\times\text{standard error} \]

The estimate gives the centre of the interval. The standard error measures sampling uncertainty. The critical value controls the confidence level.

What 95% confidence really means

The most important part of this lesson is interpretation. A 95% confidence interval does not mean there is a 95% probability that the fixed population parameter lies inside this one computed interval. In the frequentist framework, the parameter is fixed and the interval is random before the sample is observed.

SO
Sophia

That sounds subtle. If we already calculated the interval, the parameter is either inside it or not?

MR
Mr. R

Correct. The 95% refers to the long-run success rate of the method. If we repeated the sampling process many times and built intervals the same way, about 95% of those intervals would contain the true parameter.

EM
Emma

So confidence is about the procedure, not about magical probability attached to this one interval?

MR
Mr. R

Exactly. In practice we often speak informally, but mathematically the long-run interpretation is the correct one.

Width of a confidence interval

The width of a confidence interval depends mainly on three things: the variability in the data, the sample size, and the confidence level.

ChangeEffect on interval widthReason
Larger standard deviationWider intervalMore variability in individual observations
Larger sample sizeNarrower intervalSmaller standard error
Higher confidence levelWider intervalLarger critical value
OL
Oliver

So a 99% confidence interval is wider than a 95% interval?

MR
Mr. R

Yes. To be more confident in the long-run sense, the method must cast a wider net.

Confidence interval for a mean

If the population standard deviation \(\sigma\) is known and the sampling distribution is approximately normal, a confidence interval for \(\mu\) is:

\[ \bar{x}\pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}} \]

If \(\sigma\) is unknown, we use the sample standard deviation \(s\), and usually a \(t\)-critical value:

\[ \bar{x}\pm t_{\alpha/2,n-1}\frac{s}{\sqrt{n}} \]
JA
James

So the \(t\)-interval is more realistic because we usually do not know \(\sigma\)?

MR
Mr. R

Exactly. In real data analysis, \(\sigma\) is usually unknown, so the \(t\)-interval is common for means.

Confidence interval for a proportion

For a sample proportion \(\hat{p}\), a simple large-sample confidence interval is:

\[ \hat{p}\pm z_{\alpha/2}\sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

This is based on the approximate normal sampling distribution of \(\hat{p}\). The approximation works better when the sample size is large and the proportion is not too close to 0 or 1.

Confidence interval is not proof

A confidence interval gives a range of values compatible with the data and method. It does not prove all values in the interval are equally likely. It does not prove values outside are impossible. It also does not repair biased sampling or poor measurement.

SO
Sophia

So if the study design is biased, a narrow confidence interval can still be misleading?

MR
Mr. R

Yes. Confidence intervals quantify random sampling uncertainty under assumptions. They do not automatically quantify every possible source of error.

Key takeaways from this lecture
  • A confidence interval is an estimate plus a margin of error.
  • Most intervals have the form estimate ± critical value × standard error.
  • A 95% confidence level refers to long-run coverage of the procedure.
  • Higher confidence gives wider intervals.
  • Larger sample size gives narrower intervals through smaller standard error.
  • Mean intervals use \(z\) or \(t\) critical values depending on whether \(\sigma\) is known.
  • Confidence intervals quantify sampling uncertainty, not all possible errors.
Detailed theoretical notes

Confidence intervals

1. Point estimation and interval estimation

A point estimate gives a single number for an unknown parameter. An interval estimate gives a range of plausible values, acknowledging sampling uncertainty.

Confidence interval

A confidence interval is a random interval constructed from sample data using a method designed to cover the true parameter with a specified long-run frequency.

2. General confidence interval structure

Many confidence intervals have the form:

\[ \text{estimate}\pm \text{critical value}\times \text{standard error} \]

The margin of error is:

\[ \text{margin of error} = \text{critical value}\times \text{standard error} \]

3. Confidence level and significance level

If the confidence level is \(1-\alpha\), then \(\alpha\) is the total tail probability left outside the central interval.

\[ \text{confidence level}=1-\alpha \]

For a 95% confidence interval:

\[ 1-\alpha=0.95 \quad\Rightarrow\quad \alpha=0.05 \]

For a two-sided interval, the error probability is split across two tails:

\[ \frac{\alpha}{2}=0.025 \]

4. Why the 95% z-critical value is 1.96

Let \(Z\sim N(0,1)\). For a two-sided 95% confidence interval, we need the central 95% of the standard normal distribution:

\[ P(-z_{\alpha/2}\leq Z\leq z_{\alpha/2})=1-\alpha \]

For \(\alpha=0.05\):

\[ P(-1.96\leq Z\leq 1.96)\approx 0.95 \]

5. Confidence interval for a mean when \(\sigma\) is known

Suppose \(X_1,\ldots,X_n\) are independent observations from a population with mean \(\mu\) and known standard deviation \(\sigma\). From the sampling distribution:

\[ \bar{X}\approx N\left(\mu,\frac{\sigma^2}{n}\right) \]

Standardising:

\[ Z=\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}\approx N(0,1) \]

For a \(100(1-\alpha)\%\) interval:

\[ P\left(-z_{\alpha/2}\leq \frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \leq z_{\alpha/2}\right)\approx 1-\alpha \]

Solving this inequality for \(\mu\) gives:

\[ \bar{X}-z_{\alpha/2}\frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{X}+z_{\alpha/2}\frac{\sigma}{\sqrt{n}} \]

Therefore, the confidence interval is:

\[ \bar{x}\pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}} \]

6. Confidence interval for a mean when \(\sigma\) is unknown

In practice, \(\sigma\) is usually unknown. We estimate it using the sample standard deviation \(s\). This adds extra uncertainty, especially for small samples. The normal critical value is replaced by a \(t\)-critical value:

\[ \bar{x}\pm t_{\alpha/2,n-1}\frac{s}{\sqrt{n}} \]

The degrees of freedom are:

\[ \nu=n-1 \]

As \(n\) increases, the \(t\)-distribution becomes very close to the standard normal distribution.

7. Confidence interval for a proportion

Let \(X\sim \operatorname{Binomial}(n,p)\). The sample proportion is:

\[ \hat{p}=\frac{X}{n} \]

For large samples:

\[ \hat{p}\approx N\left(p,\frac{p(1-p)}{n}\right) \]

Since \(p\) is unknown, we estimate the standard error using \(\hat{p}\):

\[ \widehat{\operatorname{SE}}(\hat{p}) = \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

The large-sample confidence interval is:

\[ \hat{p}\pm z_{\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

8. Width of a confidence interval

For a mean with known \(\sigma\), the full width is:

\[ \text{width} = 2z_{\alpha/2}\frac{\sigma}{\sqrt{n}} \]

This shows the interval becomes wider when \(z_{\alpha/2}\) or \(\sigma\) increases, and narrower when \(n\) increases.

9. Interpretation of a confidence interval

The correct frequentist interpretation is:

Correct interpretation

If we repeated the same sampling procedure many times and constructed confidence intervals in the same way, approximately \(100(1-\alpha)\%\) of those intervals would contain the true parameter.

After a particular interval has been computed, the true parameter is either inside the interval or not. The confidence level describes the method, not a probability attached to this one fixed interval.

10. Common mistakes

  • Wrong: There is a 95% probability that the parameter lies in this interval.
  • Better: The method used has 95% long-run coverage under the assumptions.
  • Wrong: Values inside the interval are equally likely.
  • Better: The interval gives a range of values compatible with the data and method.
  • Wrong: A narrow interval means the study is unbiased.
  • Better: A narrow interval means high precision, not necessarily validity.
  • Wrong: A 99% interval is narrower than a 95% interval.
  • Better: A 99% interval is wider because it uses a larger critical value.
Notes summary
  • Confidence intervals extend point estimates by adding uncertainty.
  • The margin of error is critical value times standard error.
  • The 95% z-critical value is approximately 1.96.
  • For means, use \(z\) when \(\sigma\) is known and \(t\) when \(\sigma\) is estimated.
  • For proportions, use \(\hat{p}\pm z_{\alpha/2}\sqrt{\hat{p}(1-\hat{p})/n}\) as a large-sample interval.
  • Confidence level refers to long-run coverage of the procedure.

Practice exercises

Confidence intervals

Exercise 1
Basic margin of error
Easy+

An estimate is 42 and its standard error is 3. Calculate an approximate 95% confidence interval using 1.96.

Answer

Margin of error \(=1.96(3)=5.88\). The interval is \(42\pm5.88=(36.12,47.88)\).

Exercise 2
Mean interval with known sigma
Easy+

Suppose \(\bar{x}=100\), \(\sigma=20\), and \(n=100\). Construct a 95% confidence interval for \(\mu\).

Answer

\(\operatorname{SE}=20/\sqrt{100}=2\). The margin is \(1.96(2)=3.92\). The interval is \((96.08,103.92)\).

Exercise 3
Mean interval with unknown sigma
Medium+

A sample has \(\bar{x}=15\), \(s=4\), \(n=25\). Using \(t=2.064\), construct a 95% confidence interval.

Answer

\(\operatorname{SE}=4/\sqrt{25}=0.8\). Margin \(=2.064(0.8)=1.6512\). The interval is approximately \((13.35,16.65)\).

Exercise 4
Proportion interval
Medium+

In a sample of 400 students, 120 report sleeping less than 6 hours. Construct an approximate 95% confidence interval for the population proportion.

Answer

\(\hat{p}=120/400=0.30\). \(\operatorname{SE}=\sqrt{0.30(0.70)/400}=\sqrt{0.000525}\approx0.0229\). Margin \(=1.96(0.0229)\approx0.0449\). Interval: \((0.255,0.345)\).

Exercise 5
Effect of confidence level
Medium+

For the same data, which interval is wider: 90%, 95%, or 99%? Explain.

Answer

The 99% interval is widest because it uses the largest critical value. Higher confidence requires a wider interval to achieve higher long-run coverage.

Exercise 6
Interpretation
Hard+

A report says, “There is a 95% probability that the true mean is between 10.2 and 13.8.” Rephrase this more carefully.

Answer

A more careful statement is: using a method with 95% long-run coverage, we obtained the interval 10.2 to 13.8. The interval gives plausible values for the population mean under the assumptions.

Exercise 7
Interval width and sample size
Hard+

If all else is fixed, what happens to the width of a confidence interval for a mean when sample size is multiplied by 4?

Answer

The standard error is divided by \(\sqrt{4}=2\). Since the width is proportional to the standard error, the interval width is halved.

Exercise 8
Bias and confidence intervals
Hard+

Explain why a narrow confidence interval from a biased sample may be misleading.

Answer

The narrow interval indicates high precision under the sampling model, but if the sample is biased, the interval may be centred around the wrong target. Precision does not guarantee validity.
