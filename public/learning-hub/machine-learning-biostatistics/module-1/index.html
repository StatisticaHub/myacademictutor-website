Module 1

Foundations

This module introduces the core ideas behind machine learning in biostatistics: prediction, inference, validation, overfitting, data leakage and generalisation.

Start Lesson 1 Open Lesson 2

Module aim

Build the statistical foundation before using algorithms.

Before learning advanced models, students need to understand what a prediction model is, how it differs from causal explanation, and why validation is essential in medical research.

Lesson 1

What is Machine Learning in Biostatistics?

Learn what machine learning means in medical statistics and why prediction models must be valid, interpretable, clinically meaningful and tested on new patients.

Lesson 2

Validation, Overfitting and Data Leakage

Understand why models can look excellent in training data but fail in new patients, and how leakage can make performance appear unrealistically strong.

Open Lesson 2

Learning outcomes

After this module, you should be able to explain:

Prediction vs inference

Why predicting an outcome is not the same as explaining or estimating a causal effect.

Validation

Why models must be tested on unseen data before being trusted in medical settings.

Data leakage

How future information or incorrectly processed predictors can make a model look falsely strong.
