Machine Learning in Biostatistics

A structured course on machine learning for medical statistics, clinical prediction modelling, regularisation, tree-based methods, high-dimensional biomedical prediction, survival analysis, and biomedical data science.

Learn how to build, validate, interpret and report prediction models in medical and biomedical research using careful statistical thinking and reproducible R workflows.

Course progress

Modules 1–5 are now available.

The course currently covers foundations, supervised learning, model evaluation, validation, calibration, clinical usefulness, regularisation, feature selection, high-dimensional penalised regression, decision trees, random forests, gradient boosting, XGBoost, variable importance and partial dependence.

Important idea: In medical machine learning, high accuracy is not enough. A useful model should be valid, calibrated, clinically interpretable, reproducible, clinically useful, and generalisable to new patients or populations.

Prediction and validation

Understand prediction versus inference, overfitting, data leakage, train-test splitting, internal validation and generalisation.

Clinical performance

Learning pathway

A recommended way to use this course.

1. Read the lecture

Start with the story-based lecture to understand intuition, doubts and clinical motivation.

2. Study the notes

Use the notes section inside each lesson for deeper theory, statistical explanation and interpretation.

3. Complete exercises

Test conceptual understanding and applied reasoning before moving to implementation.

4. Run the R code

Implement the method and interpret model outputs in a reproducible workflow.

5. Work through case studies

Connect the method to realistic medical prediction problems with R scripts, figures, interpretation and limitations.

6. Reflect clinically

Ask whether the model is valid, calibrated, useful and safe for the intended setting.
