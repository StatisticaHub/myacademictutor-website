A structured course on machine learning for medical statistics, clinical prediction modelling, survival analysis, and biomedical data science.

Course overview

Machine learning through a medical statistics lens.

This course is designed for learners who want to understand machine learning through a biostatistics, clinical prediction and biomedical data science perspective.

Important idea: In medical machine learning, high accuracy is not enough. A useful model should be valid, calibrated, clinically interpretable, reproducible, clinically useful, and generalisable to new patients or populations.

Prediction and validation

Understand prediction versus inference, overfitting, data leakage, train-test splitting, internal validation and generalisation.

Clinical performance

Learn discrimination, calibration, Brier score, decision curves, sensitivity, specificity and clinical usefulness.

Reproducible R workflows

Use R scripts and applied case studies to connect modelling methods to practical medical prediction workflows.

Learning pathway

A recommended way to use this course.

1. Read the lecture

Start with the story-based lecture to understand intuition, doubts and clinical motivation.

2. Study the notes

Use the notes for deeper theory, statistical explanation and interpretation.

3. Complete exercises

Test conceptual understanding and applied reasoning before moving to implementation.

4. Run the R code

Implement the method and interpret model outputs in a reproducible workflow.

5. Work through case studies

Connect the method to realistic medical prediction problems with figures and interpretation.

6. Reflect clinically

Ask whether the model is valid, calibrated, useful and safe for the intended setting.
