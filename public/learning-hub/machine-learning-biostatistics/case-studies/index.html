Applied medical machine learning

Case Studies

Case studies connect the course methods to complete medical prediction workflows: clinical question definition, outcome and predictor choice, model fitting, validation, calibration, threshold interpretation, figures, limitations and reproducible R code.

The aim is to show that medical machine learning is not only about fitting models. A useful workflow must define the prediction question clearly, avoid leakage, evaluate discrimination and calibration, interpret thresholds carefully, and explain limitations honestly.

Case Study 1

Diabetes Risk Prediction

Available Logistic regression ROC/AUC Calibration

A complete logistic regression clinical prediction workflow using the Pima Indians Diabetes dataset. This case study introduces the full process from prediction question to clinical interpretation.

  • binary outcome definition;
  • train/test split;
  • logistic regression and odds ratios;
  • confusion matrix;
  • sensitivity, specificity, PPV and NPV;
  • ROC curve and AUC;
  • Brier score and calibration plot;
  • threshold interpretation;
  • clinical limitations and external validation.

Case Study 2

Regularised Diabetes Risk Prediction

Available Ridge Lasso Elastic net

Extends Case Study 1 by comparing ordinary logistic regression with ridge regression, lasso regression and elastic net. The focus is shrinkage, feature selection, validation, calibration and careful interpretation.

  • ordinary logistic regression baseline;
  • ridge coefficient shrinkage;
  • lasso feature selection;
  • elastic net as a compromise between ridge and lasso;
  • AUC and Brier score comparison;
  • ROC curve comparison;
  • selected-feature interpretation;
  • cross-validation of penalty strength;
  • why selected features are not automatically causal.

How to use the case studies

A practical workflow for learners.

1

Read the question

Start by understanding the clinical prediction question and prediction time point.

2

Run the R script

Generate the model output and figures from the reproducible script.

3

Interpret results

Study AUC, Brier score, calibration, thresholds and clinical meaning.

4

Reflect critically

Ask what could go wrong before any model could be used in practice.

Key message

Case studies are about judgement, not only code.

A useful clinical prediction workflow must define the prediction problem clearly, use predictors available at the prediction time point, avoid leakage, validate performance honestly, examine calibration, choose thresholds based on clinical purpose, and report limitations transparently.
