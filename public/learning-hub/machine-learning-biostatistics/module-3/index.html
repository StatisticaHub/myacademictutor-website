Module aim

Move beyond apparent accuracy.

A medical prediction model must be evaluated carefully. This module focuses on whether a model generalises, whether probabilities are reliable, and whether predictions can support decisions.

Key ideas

What you will learn to judge.

Discrimination

How well the model separates patients with and without the outcome.

Calibration

Whether predicted probabilities are numerically close to observed risks.

Clinical usefulness

Whether using the model leads to better decisions than simple alternatives.
