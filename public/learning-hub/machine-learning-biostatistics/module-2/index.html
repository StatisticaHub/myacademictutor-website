Module aim

Understand prediction models for binary medical outcomes.

Many clinical prediction problems involve binary outcomes, such as disease present or absent, treatment response or no response, readmission or no readmission. This module explains the basic supervised learning workflow.

Key ideas

What you will practise in this module.

Outcome definition

Define the clinical event clearly and understand what the model is trying to predict.

Predicted probabilities

Interpret model outputs as risks or probabilities rather than only labels.

Medical metrics

Use metrics that match the clinical decision, not only overall accuracy.
